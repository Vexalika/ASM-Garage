Arbitrarily define weapons that are Modern Brave(Only x2 attacks when initiating combat). 

This has pretty straight foward installation and usage.

Installation
Include the .event file somwhere in free space and add "ContemporaryBrave" to the Skill System PreBattle Calc Loop

Usage
Weapons in the "ModernBraveWeaponList" will act as modern brave weapons. By default all vanilla brave weapons are included remove them to have them function as vanilla GBA brave weapons.
